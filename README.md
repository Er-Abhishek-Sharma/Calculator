# 🧮 Streamlit Calculator App

A simple, clean, and interactive calculator built using Python and Streamlit.
This app performs basic arithmetic operations with real-time results and keeps a history of calculations.

---

## 🚀 Features

* ⚡ Real-time calculation (no page reload)
* ➕ ➖ ✖ ➗ Basic arithmetic operations
* 📱 Clean and responsive UI
* 🧠 Session-based calculation history
* 🧹 Clear history option
* ❌ Error handling (division by zero)

---

## 🛠️ Built With

* Python
* Streamlit

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator-app.git
cd calculator-app
```

2. Install dependencies:

```bash
pip install streamlit
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

The app will open automatically in your browser.

## 🧠 How It Works

* Uses Streamlit widgets like `number_input`, `radio`, and `button`
* Stores history using `st.session_state`
* Updates results instantly when inputs change
